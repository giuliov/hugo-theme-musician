## Setup

```Powershell
cd C:\src\github.com\rebyv\site\
$hugoVersion = '0.100.1'
Set-Alias hugo "C:\bin\hugo\${hugoVersion}\hugo.exe"
#$env:PATH += ";C:\bin\sass_embedded;"
hugo serve
```
